# Cron Explainer — Cron 表达式解析

Clipbus 插件，自动检测剪贴板中的 cron 表达式并可视化分解每个字段。

## 功能

- **自动检测**：识别标准 5 字段 cron 表达式（分钟 时 日 月 周）
- **字段分解**：每个字段的原始值与人类可读说明以表格形式展示
- **自然语言摘要**：末尾一句话描述调度周期
- **范围校验**：分钟 0–59、小时 0–23、日 1–31、月 1–12（支持 JAN–DEC）、周 0–7（支持 SUN–SAT）
- **误判防护**：纯数字字符串（如 `1 2 3 4 5`）被忽略——至少需含一个 cron 特殊字符（`* / , -`）

## 支持的语法

| 语法 | 示例 | 含义 |
|---|---|---|
| 通配符 | `*` | 每 … |
| 整数 | `30` | 第 30 … |
| 范围 | `1-5` | 1 到 5 |
| 列表 | `1,3,5` | 枚举值 |
| 步长 | `*/15` | 每 15 … |
| 月份名 | `JAN-MAR` | 一月到三月 |
| 周几名 | `MON-FRI` | 周一到周五 |

## 示例

```
30 9 * * 1-5     # 工作日 9:30
*/15 * * * *     # 每 15 分钟
0 0 1 * *        # 每月 1 日零点
0 8 * * MON-FRI  # 工作日 8:00
```
