# XML Formatter Plugin

自动识别剪贴板中的 XML / HTML 片段，进行格式化缩进并展示结构统计。

## 功能

- **XML 识别**：检测以 `<` 开头、`>` 结尾且含 2 个以上标签的文本片段（兼容 `<?xml?>` 声明、注释、自闭合标签）；拒绝 JSON 与纯文本
- **格式化展示**：按层级重新缩进（每层 2 空格），支持滚动查看大文件
- **结构统计**：卡片顶部显示元素数、属性数、最大嵌套深度
- **一键复制**：点击"复制格式化 XML"按钮将格式化结果写入剪贴板
- **快捷 Action**：`xml-copy` 自动运行动作，直接输出格式化文本；非 XML 输入返回 none

## 能力清单

| 能力 | ID | 说明 |
|------|-----|------|
| Detector | `xml-detector` | 识别 XML/HTML 片段 |
| Renderer | `xml-renderer` | 格式化卡片，含统计信息 |
| Action | `xml-copy` | 输出格式化 XML 文本 |
