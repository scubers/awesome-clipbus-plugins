# clipbus-radix-plugin — 进制转换

自动识别整数字面量并在四种进制之间相互转换。

## 功能

- **自动检测** 剪贴板中的整数字面量，支持十进制、十六进制（`0x`）、二进制（`0b`）、八进制（`0o`）四种格式
- **任意大整数** 使用 BigInt 解析，无精度丢失
- **卡片展示** 四行对照显示 DEC / HEX / OCT / BIN，每行可单独点击"复制"
- **元信息** 显示位数（bits）及可打印 ASCII 字符（仅 32–126 范围）
- **一键复制** 卡片按钮"复制全部进制"，生成 `DEC: …\nHEX: …\nOCT: …\nBIN: …` 多行文本
- **Auto-Run Action** `radix-copy`：复制到剪贴板时直接返回四进制文本；非整数输入返回 none

## 示例

| 输入 | DEC | HEX | OCT | BIN |
|---|---|---|---|---|
| `255` | 255 | 0xff | 0o377 | 0b11111111 |
| `0xff` | 255 | 0xff | 0o377 | 0b11111111 |
| `0b1010` | 10 | 0xa | 0o12 | 0b1010 |
| `0o777` | 511 | 0x1ff | 0o777 | 0b111111111 |
