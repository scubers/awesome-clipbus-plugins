# clipbus-markdown-plugin — Markdown Preview

Detects **Markdown text** on the clipboard and renders it locally as a safe HTML card.

## Features

- **Markdown Detector** (`markdown-detector`): Counts distinct Markdown signal types in the text (ATX headings, bold/italic, lists, links, fenced code blocks, inline code, blockquotes) and triggers only when **at least 2 signal kinds** are present, avoiding false positives on ordinary prose.
- **Markdown Preview Card** (`markdown-renderer`):
  - Meta bar at the top showing character count, line count, and heading count.
  - Renders a safe subset: h1–h6, `**bold**` / `*italic*` / `_italic_`, inline `` `code` ``, fenced code blocks, unordered/ordered lists, blockquotes, and `[link](url)`.
  - Adaptive height (min 120 / max 480).

## Security

- **Only tags generated by this plugin are rendered.** Source text is HTML-escaped (`& < > "` → entities) before Markdown patterns are applied — no raw HTML is passed through.
- Link `href` values are restricted to `http://`, `https://`, and `mailto:` protocols. Unsafe schemes such as `javascript:` are demoted to plain text and never executed.
- All rendering happens locally; no network requests are made.

## Development

```sh
npm install
npm run verify   # typecheck + lint + build + verify-build + tests
```
